-----------------------------------------------------------------------
Requirements
-----------------------------------------------------------------------
Requires OpenGL drivers to display Preview of textures. Program will
still function without OpenGL present, you simply won't have a preview.

-----------------------------------------------------------------------
Installation
-----------------------------------------------------------------------
Copy 'D3 Image Tool.exe' and 'D3ITHelp.chm' to any directory you wish.
They must be located in the same place or else D3 Image Tool will not
be able to load the Help System.
